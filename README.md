# Karşılama Projesi 👋

Bu basit web projesi, kullanıcıdan isim alır ve giriş yaptıktan sonra kişiye özel bir karşılama ekranı sunar. Ekranda o anki **saat** ve **gün** bilgisi gösterilir ve her saniye güncellenir.

## 🚀 Özellikler

- İsim giriş ekranı
- “Hoş geldin” mesajı
- Canlı saat ve gün bilgisi
- Zamanla değişen arka plan (isteğe bağlı)
- Emoji destekli mesajlar 😊

## 📁 Proje Dosyaları

- `index.html` – HTML yapısı
- `style.css` – Sayfa tasarımı
- `script.js` – JavaScript kodları
- `README.md` – Proje açıklaması
